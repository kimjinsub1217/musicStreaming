# 📌 musicStreaming
> 음악스트리밍

## ✔기능
<ul>
  <li>Retrofit 을 이용하여 재생 목록을 받아옴  </li>
  <li>재생 목록을 클릭하여 ExoPlayer 를 이용하여 음악을 재생함 </li>
  <li>PlayList 화면과 Player 화면 간의 전환을 할 수 있음.</li>
</ul>

##  ✔활용 기술
MotionLayout, ExoPlayer, Retrofit
